## 🔧 Technical Documentation

### Backend Architecture
- **Framework:** FastAPI (async, high-performance)
- **Database:** SQLite + SQLAlchemy ORM
- **ML Integration:** PyTorch models served via HTTP
- **Authentication:** JWT + OTP (printed to terminal in demo)
- **Document:** README_COMPLETE.md + API_DOCUMENTATION.md

### Frontend Stack
- **Frameworks:** React 18 + Vite (both frontends)
- **Rider PWA:** Port 3201 (gig workers interface)
- **Insurer Dashboard:** Port 3001 (admin interface)
- **HTTP Client:** Axios with interceptors
- **Document:** SETUP_GUIDE.md + README_COMPLETE.md

### ML Models
- **Premium Model:** PyTorch neural network (personalized pricing)
- **Forecast Model:** Time series predictions
- **CNN Model:** GPS fraud verification
